# MECM-Distribution-Groups

 Follow the steps below to create distribution groups in Microsoft Endpoint Configuration Manager.
  
- In the Administration tab click on Distribution Point Groups.
- Click on Create Group.
- Click Add Distribution Points
  
<p align="center">
<img src="https://i.imgur.com/Adtx3AC.png" height="80%" width="80%"/>

- Select the desired server which MECM is associated with in your domain.
- Click OK.

<p align="center">
<img src="https://i.imgur.com/pCNbp3E.png" height="80%" width="80%"/>

- Click the Collections Tab.
- Click Add.

<p align="center">
<img src="https://i.imgur.com/10l1Lf1.png" height="80%" width="80%"/>

- Select Device Collections.
- Check off the collection you want included.
- Click OK

<p align="center">
<img src="https://i.imgur.com/DduqyMo.png" height="80%" width="80%"/>

- Add Name to Distribution Group.
- Click OK.

<img src="https://i.imgur.com/0y11xRW.png" height="80%" width="80%"/>
